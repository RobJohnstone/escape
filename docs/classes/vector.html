vector Class

Defined in: js/vector.js:1
Module: vector

A library of useful vector math functions

At first glance it may appear that a nicer syntax could be obtained by creating a vector type and attaching these methods to the prototype. However, the most common use case is an object of a different type that happens to have x and y properties. As using apply/call is fiddly in terms of syntax and multiple inheritance is memory inefficient in javascript (an object can only be linked to a single prototype), I believe this approach to be best.
